Easy to add a link to application and extension on your post

== Description ==

SimpleAppLinks displays link to application and extension on your WordPress
posts and pages.

Supported platform is below...

* Google Play
* iTunes App Store
* Mac App Store
* Windows Store
* Chrome Web Store
* Firefox Add-ons
* Wordpress Plugins

= How to Use = 

SimpleAppLinks uses 'applink' shortcode with url parameter. Like this...

[applink
url="https://play.google.com/store/apps/details?id=jp.loumo.paydaycounter"]

== Installation ==

1. Upload simpleapplinks to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
